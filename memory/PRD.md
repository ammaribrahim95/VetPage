# KLINIK HAIWAN TTDI VET - Website Project

## Original Problem Statement
Create a modern, clean, and trustworthy veterinary clinic website for KLINIK HAIWAN TTDI VET located in Malaysia.

**Brand Tone:** Caring, professional, gentle, friendly, modern, and community-focused  
**Target Audience:** Pet owners (cats & dogs), families, young professionals, and nearby residents

## User Personas
1. **Pet Owner (Primary)** - Local residents with cats/dogs needing veterinary care
2. **First-time Visitors** - People searching for a trusted vet in TTDI area
3. **Emergency Cases** - Pet owners needing immediate contact/directions

## Core Requirements (Static)
- Professional landing page design
- Mobile-responsive layout
- WhatsApp booking integration
- Google Maps integration for directions
- Showcase veterinarians Dr. Jeanne and Dr. Boey
- Display 8 real customer testimonials
- Clean green-themed design (Green-AI guidelines)
